### **2) Configure MEGA Credentials**

Open **`mega.js`** and replace the example values with your own account details.

```js

const auth = {
  email: 'your-email@example.com',
  password: 'your-password',
  userAgent: 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/137.0 Safari/537.36'
};
```
