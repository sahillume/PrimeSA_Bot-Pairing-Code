# 📥 Installation

Clone the repository

```bash
git clone https://github.com/sahillume/PrimeSA_Bot-Pairing-Code.git
```

Open the project

```bash
cd PrimeSA_Bot-Pairing-Code
```

Install packages

```bash
npm install
```

Start the bot

```bash
npm start
```

---
